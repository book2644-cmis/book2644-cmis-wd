How to create a new webpage on the production server?

  1. In your repository folder, create a file called index.html . Type gedit index.html to create a file.
  2. Type the basic code of html. This will be a structure/base to your website. Save your edit.
  3. Create another file called main.css . Save.
  4. Copy this text: link rel="stylesheet" href="main.css" type="text/css" .Paste it in your index.html file.
  5. Type in index.html if you wanted text to appear on your webpage. Decorate in main.css. Save both your changes.
  6. When you finished, type sftp username@cmis-hsteacher .
  7. Hit yes, then type in your password.
  8. Type " cd public_html ", then type " put index.html ", repeat this step with main.css file.
  9. Type " exit "
  10. Type " git add index.html " Enter.
  11. Type " git commit -m "descrpition" " Enter.
  12. Type " git push " Enter.
  13. Type in your username then your password. This will save your file in github.
  14. Add your main.css file into github as well.
